## Getting Started

*Install bundler*

```shell
gem install bundler
```

*Install ruby gems*

```shell
bundle install
```

*Install node packages*

```shell
yarn
```

*Start the Middleman server*

```shell
bundle exec middleman
```

> Located at `localhost:4567`
